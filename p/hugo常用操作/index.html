Hugo常用操作

Sep 07, 2021
阅读时长: 1 分钟

  1. 用cmd指令创建博客原空间，在myblog中会生成一系列文件

     hugo new  site  myblog(空间名)

  2. 通过gitbash在 https://themes.gohugo.io/下载主题 到原空间的themes文件下到原空间的themes文件下

  3. 本地启动的时候指定主题名，访问 http://localhost:1313/

     hugo server  -t 主题名	 --buildDrafts

  1. 在post目录下创建第一篇文章

    hugo new post/test/myblog.md

  2. 把博客部署到github，这行命令会在原空间生成pubilc文件(已经存在则更新)

hugo --theme=hugo-theme-stack --baseUrl="https://huashengzheng.github.io" --buildDrafts

  1. 在public目录执行git命令

    git init
git add .
git  commit -m "第一次提交"

git  remote add origin https://github.com/huashengzheng/huashengzheng.github.io.git    
推到git
git push -f -u origin master (这个-f 谨慎用，把远程git强制更新成本地的)
需要git账号和token

7.完成后git就能看到刚刚的更新

8.访问huashengzheng.github.io就能看到博文内容了
